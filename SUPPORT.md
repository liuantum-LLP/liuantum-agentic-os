# Support

## Where to Ask Questions

- **GitHub Issues**: For bugs, feature requests, and documentation improvements.
- **Discussions**: For general questions, ideas, and community conversation.

## Before Filing an Issue

1. Run `./liuant troubleshoot` and check the output.
2. Search existing issues for similar topics.
3. Review `docs/TROUBLESHOOTING.md` for common problems.

## Filing a Bug Report

Include:
- Operating system and version
- App version (`./liuant version`)
- CLI command you ran
- Full output (with secrets redacted)
- Steps to reproduce
- Expected behavior
- Actual behavior

## Filing a Feature Request

Include:
- What problem you are trying to solve
- How you envision the feature working
- Any relevant examples from other projects
- Whether it requires external API access or new dependencies

## Security Issues

Do not file security issues publicly. See `SECURITY.md` for responsible disclosure.
