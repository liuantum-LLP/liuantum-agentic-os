# Roadmap

## Current Status (v0.8.0)

- Open-source release preparation complete.
- MIT licensed.
- Local-first, chat-first AI agent operating system.
- macOS DMG builds (unsigned), CLI, and desktop UI all working.
- 427+ tests passing.

## v0.8 — Open Source Release

- [x] MIT License
- [x] Open-source repository files (CONTRIBUTING, CODE_OF_CONDUCT, SECURITY)
- [x] CI workflow without Apple credentials
- [x] Community build model documented
- [x] Secret/privacy audit
- [x] Public README

## v0.9 — Sidecar Backend & Packaging

- Bundled sidecar executable for single-binary distribution.
- Improved installer scripts for macOS, Linux, Windows.
- Reproducible build documentation.
- Optional maintainer signing workflow.

## v1.0 — Stable Local Desktop Release

- Production-ready local desktop experience.
- Polished onboarding and first-run flow.
- Comprehensive documentation for contributors and users.
- Stable API surface.

## Future (Post-v1.0)

- More provider SDK integrations.
- RAG quality improvements.
- Additional social connectors (Meta, YouTube, Reddit).
- Plugin/skill ecosystem.
- Community governance model.

## Non-Goals

- Cloud-hosted SaaS offering.
- Multi-user tenancy (local-first always).
- Auto-publishing or auto-sending email/social.
- Apple App Store distribution.
- Windows/Linux code signing (not planned).
