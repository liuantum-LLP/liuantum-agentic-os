# Basic Python Skill Template

A minimal Python skill template for Liuant Agentic OS.

## Getting Started

1. Copy this template: `./liuant skills create my-skill --template basic-python-skill`
2. Edit `skill.json` with your skill's metadata
3. Edit `skill.py` with your skill's logic
4. Validate: `./liuant skills validate ./skills/installed/my-skill`
5. Install: `./liuant skills install ./skills/installed/my-skill`
6. Enable: `./liuant skills enable my-skill`
7. Run: `./liuant skills run my-skill --input '{"message":"hello"}'`
