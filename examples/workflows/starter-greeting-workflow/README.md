# Starter Greeting Workflow

A simple workflow that demonstrates the basics of running a skill within a workflow.

## Overview

This workflow shows how to chain a single skill (hello-skill) and format its output. It's perfect for learning workflow structure and testing your setup.

## Skills Used

- `hello-skill`: Generates a greeting message

## Installation

This workflow is part of the official Liuant workflow examples. No additional installation required.

## Usage

```bash
liuant skills workflow preview starter-greeting-workflow --input '{"user_name":"Alice"}'
liuant skills workflow permissions starter-greeting-workflow
liuant skills workflow run starter-greeting-workflow --input '{"user_name":"Alice"}'
```

## Permissions

This workflow requires no special permissions.

## Input

```json
{
  "user_name": "Alice"
}
```

## Output

The workflow produces:
- `greeting`: The greeting message generated by hello-skill

## Example

```bash
liuant skills workflow run starter-greeting-workflow --input '{"user_name":"Bob"}'
```
